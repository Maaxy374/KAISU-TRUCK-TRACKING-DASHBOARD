# KAISU Truck Tracking Dashboard

## 🚀 Live Fleet Tracking & Operations

A modern, real-time fleet tracking dashboard built with vanilla HTML, CSS, and JavaScript. Track truck positions, monitor parking alerts, and manage driver assignments from a sleek, responsive interface.

### ✨ Features

- **Real-time Fleet Tracking** - Live vehicle status and position monitoring
- **Priority Alerts** - Highest-risk conditions sorted by severity
- **Interactive Map** - Visual fleet overview with status indicators
- **Advanced Filtering** - Search and filter by status, truck number
- **Responsive Design** - Works seamlessly on desktop and mobile
- **Mock API Integration** - Ready for real backend connection
- **Auto-refresh** - Keep data current with manual refresh button

### 📊 Dashboard Sections

1. **Operations Snapshot** - Key metrics at a glance
2. **Map Overview** - Visual fleet distribution
3. **Operational Metrics** - Fleet statistics and ratios
4. **Priority Alerts** - Top issues requiring attention
5. **Fleet Vehicle Register** - Complete searchable truck list

### 🔧 Tech Stack

- **Frontend**: HTML5, CSS3 (no frameworks)
- **JavaScript**: Vanilla ES6+
- **API Layer**: Mock API (easily replaceable)
- **Styling**: Custom design system with CSS variables
- **Responsive**: Mobile-first design
